Introduction

This guide provides instructions on how to find out information about persons, entities, and things (websites, emails, files, etc.), using a large number of  Open Source Intelligence (OSINT) tools.

This document is updated from time to time. If you wish to be notified of updates, please subscribe to the mailing list.

How to use this guide

This guide is one page, divided into three sections: persons, entities, and things.  Links to these sections are in the top menu.  The topics are organized as statements describing the outcome. If you want to search the page for a particular something, hitting  CTRL-F (or Command-F on a Mac) should give you access to the search functionality of your browser.  Keep the search short (try one word first) and you should be able to find what you need quickly.
