Introduction

This guide provides an introductory roadmap for conducting investigations using a large number of (mostly free) Open Source Intelligence (OSINT) tools. It should enable someone with basic computer skills to gather meaningful information about a person, entity or many forms of electronic records, online.

It is important to state at the outset, that if any of the information gathered is to be used as evidence, it is important to consult a properly trained computer forensics professional to ensure that the continuity and integrity of the evidence is maintained, consistent with the evidentiary requirements in the applicable jurisdiction.

This document is updated from time to time. If you wish to be notified of updates, please subscribe to the mailing list.
